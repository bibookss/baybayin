Para saan ang Baybayin?

Ang Baybayin ay isang web app na naglalayong magturo ng Baybayin sa paraan ng laro gamit ang kakayahan ng Machine Learning.

Nais nitong bigyan ng espasyo sa web ang kulturang Filipino upang mapalaganap at mapreserba ito

Upang maging mas accurate ang mga machine learning models para sa Baybayin handwriting recognition, nais ng app na itong kumolekta ng karagdagang data para sa ka ikalalago ng accuracy ng model na ito.

Ano ang Baybayin?

Ang Baybayin ang dating sistemang panulat sa Pilipinas bago ang panahon ng mga Español. Laganap ito sa bansa at nag-evolve ito nang ma-adapt sa iba't ibang lugar sa Pilipinas. Sa kasalukuyan, mayroong nakikitang revival sa paggamit nito.
